# Conveyor Monitoring Dashboard

## О проекте

**Conveyor Monitoring Dashboard** — это frontend-проект дашборда для мониторинга конвейерных линий.
Приложение позволяет просматривать состояние линий, работать с инцидентами, открывать карточку отдельного инцидента и анализировать сводные данные по сменам.

Проект реализован как **SPA (Single Page Application)** на **React + TypeScript + Vite**.

---

## Цель проекта

Цель проекта — разработать понятный и удобный интерфейс мониторинга, который помогает:

- быстро оценивать текущее состояние конвейерных линий;
- видеть проблемные линии и тревожные события;
- просматривать список инцидентов;
- работать с карточкой отдельного инцидента;
- анализировать сводные данные по дате и смене.

---

## Что реализовано

### Обзор
- вывод тестовых конвейерных линий;
- отображение статуса, скорости, нагрузки, количества камер и тревог;
- фильтрация по статусам;
- визуальное выделение alarm-линий;
- приглушение offline-линий.

### Инциденты
- страница со списком тестовых инцидентов;
- отображение статуса, ответственного, комментария и даты;
- переход в карточку отдельного инцидента.

### Карточка инцидента
- детальная информация по выбранному инциденту;
- история изменения статусов;
- локальная форма изменения статуса;
- валидация: при выборе статуса `resolved` комментарий обязателен.

### Отчёт
- выбор даты;
- выбор смены;
- отображение сводной статистики;
- график событий по часам;
- график событий по линиям;
- экспорт отчёта в CSV.
