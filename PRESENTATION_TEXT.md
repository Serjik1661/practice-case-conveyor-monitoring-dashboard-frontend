# PRESENTATION_TEXT

Здравствуйте. Я представляю проект Conveyor Monitoring Dashboard.

Это frontend-приложение для мониторинга конвейерных линий. Основная цель проекта - дать оператору единый интерфейс, в котором можно видеть состояние линий, проблемные события, список инцидентов и отчётные данные по сменам.

На текущем этапе у меня реализованы основные страницы проекта:
- обзор линий;
- список инцидентов;
- карточка отдельного инцидента;
- страница отчёта.

Также в проекте реализован mock API через MSW, чтобы интерфейс работал не только с простыми локальными массивами, а через более реалистичную схему запросов. На странице инцидента можно изменять статус через mock PATCH-запрос, а на странице отчёта уже есть графики и экспорт в CSV.

Дополнительно в проекте есть базовая realtime-логика и toast-уведомления, а сам проект уже опубликован онлайн.

Сейчас проект находится на этапе рабочего MVP. То есть основная структура и ключевая логика уже готовы, а дальше проект можно дорабатывать в сторону более полного backend, улучшения realtime и общей полировки интерфейса.

На текущий момент проект уже можно показывать как готовую учебную frontend-работу.
