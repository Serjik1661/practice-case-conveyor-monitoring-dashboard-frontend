# AGENTS.md

## Коротко о проекте
Учебный frontend‑проект дашборда мониторинга конвейерных линий.
Стек: React, TypeScript, Vite, React Router, Recharts.

## Структура проекта (кратко)
- `src/` — весь код приложения.
- `src/pages/` — страницы (OverviewPage, IncidentsPage, IncidentPage, ReportPage).
- `src/shared/` — общие типы и утилиты.
- `src/mocks/` — фикстуры и MSW‑хендлеры для mock API.
- `public/` — статика и сервис‑воркер MSW.

## Команды
- `npm run dev` — запуск в режиме разработки.
- `npm run build` — сборка проекта.
- `npm run preview` — локальный просмотр собранной версии.
- `npm run lint` — проверка линтера.

## Правила работы
- Объяснять, что делается и зачем, простыми словами.
- Менять минимально необходимое количество файлов.
- Не делать больших рефакторингов без явной необходимости.
- Сохранять TypeScript аккуратным и понятным.
- После каждой задачи перечислять изменённые файлы.
- Писать решение по шагам и простыми словами.

## Готовый результат (Definition of Done)
- Задача выполнена и работает в браузере.
- Текущие страницы не сломаны.
- Импорты и типы TypeScript корректны.
- Список изменённых файлов указан.

## Проверка локально
1. Установить зависимости: `npm install`
2. Запустить приложение: `npm run dev`
3. Открыть в браузере адрес, который покажет Vite.
4. Проверить нужную страницу и сценарий.
